# GulfTagAE (GEMS Education x Startopia 2026)
This GitHub Repository contains the source code to my solution (The GulfTag AE website.) 

GulfTag AE is a simple system I built to help students, parents, and schools keep track of lost items. Many students lose bottles, lunch boxes, hoodies, and other belongings every day, and most of the time there is no way to know who the owner is. GulfTag AE helps identify items quickly using QR codes.

I have used JavaScript (script.js), HTML (index.html), and CSS (style.css).
This is the link to my website : https://thefakepepsi.github.io/GulfTagAE/
